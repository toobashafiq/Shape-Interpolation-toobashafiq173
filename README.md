# Shape Interpolation

This is a simple web project where you can draw two shapes and see a smooth animation between them.

---
**Live Demo**
https://shape-interpolation-173.vercel.app

## Features

* Draw Shape A (blue)
* Draw Shape B (red)
* Animate between shapes
* Reset anytime

---

## Controls

* Left Click → Draw shape
* Spacebar → Switch to Shape B
* S → Start/Stop animation
* R → Reset
